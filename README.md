```python

    audio_url="string_example",

    model="sync-1.5.0",
)
```

##### audio_url: `str`<a id="audio_url-str"></a>

##### video_url: `str`<a id="video_url-str"></a>

A url to the video file to be synchronized -- must be publicly accessible
